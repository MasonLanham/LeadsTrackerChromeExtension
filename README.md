Built as part of the Full-Stack Developer Path on Scrimba for the JavaScript Essentials Course.

To Deploy this to your Chrome-based browser:

1) Download the files and place them all in the same folder.
2) Enter the Chrome://extensions url in your search bar.
3) Turn Developer Mode On
4) Select "Load Unpacked"
5) Select the folder containing the files for this extension.
6) You're done that should install the extension!
